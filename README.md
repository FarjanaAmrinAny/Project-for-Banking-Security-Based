# Project-for-Banking-Security-Based
Secure Banking Transaction Monitoring System
Live Features:

🔴 Real-time transaction table with simulated live feed (auto-refreshes every ~4s)
➕ Simulate button to inject new transactions manually
🔍 Search/filter by TXN ID, account name, or transaction type
📋 Transaction detail modal with full info, activity log, and Approve/Block actions

Monitoring Panels:

KPI cards — volume, transaction count, flagged/blocked, false positive rate
Velocity chart — animated area chart with anomaly spike overlay (switchable 1H/6H/24H/7D)
System Risk Index dial — visual gauge showing current threat level
Alert feed — ranked critical/warning/info alerts with real-time updates
Geographic activity map — hotspot visualization with risk zones
ML Model stats — Precision, Recall, F1, and latency metrics

Risk Classification:
Each transaction is scored 0–99 by a simulated ML engine and labeled HIGH / MED / LOW with color-coded badges and status (APPROVED / FLAGGED / BLOCKED / REVIEW
